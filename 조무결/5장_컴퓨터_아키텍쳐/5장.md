- Ram16k(8k 16bit RAM) = 16 * 1024 레지스터 램 
- 메모리 맵 : 램과 픽셀을 맵핑 
- CPU가 메모리를 변경하면 SCREEN에 바로 반영
- 스크린 : 131,072 픽셀 (512*256)
- 메모리 : 131072개의 비트 데이터 저장 가능 (8*1024*16)
- 픽셀 메모리 주소 계산 = RAM[r*32+c/16)] => r*(가로/16) + c/(세로/16)  = c % 16
